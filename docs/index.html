Indian Food Database API

This is a simple API for Indian food nutritional data from IFCT 2017.

Endpoints:

  • /search-index.json - Search index with basic food information
  • /all-foods.json - Complete database with all food items
  • /foods/{id}.json - Individual food details by ID

Sample Data:

Field Description
id Unique identifier for the food item (based on food name)
name Name of the food item
caloriesPer100g Energy in kcal per 100g
proteinPer100g Protein content in grams per 100g
carbsPer100g Carbohydrate content in grams per 100g
fatPer100g Fat content in grams per 100g

Total foods in database: 1014

Data source: IFCT 2017 (Indian Food Composition Tables)
